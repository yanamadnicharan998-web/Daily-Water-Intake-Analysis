# Daily Water Intake Analysis - Data Mining Project

## HOW TO RUN
1. Go to colab.research.google.com
2. File -> Upload Notebook -> select Daily_Water_Intake_Analysis.ipynb
3. Run Step 2 cell -> it will ask you to UPLOAD the Excel file
4. Upload: dataset/water_intake_dataset.xlsx
5. Runtime -> Run All -> All 11 graphs appear!

## Dataset
- File: dataset/water_intake_dataset.xlsx
- Source: Kaggle
- URL: https://www.kaggle.com/datasets/sonalshinde123/daily-water-intake-and-hydration-patterns-dataset
- Records: 100 | Features: 16

## Project Files
- Daily_Water_Intake_Analysis.ipynb  <- Main notebook (reads from Excel)
- dataset/water_intake_dataset.xlsx  <- Excel dataset (show to faculty)
- output_graphs/                     <- Graphs saved here after running
